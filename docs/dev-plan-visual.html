关键技术决策

决策点选择理由
排版库赫蹏 (heti)内置古文/竖排/行间注/诗词版式，MIT，6.4k stars
拼音生成pinyin-pro (Node.js)已有依赖，API 简单，支持声调标注
前端框架无（原生 JS）项目规模小，无需框架复杂度
数据传递构建 JSON + 运行时 fetch分离构建与渲染，页面可离线使用
路由方案URL hash (#poem-001)无需服务端配置，GitHub Pages 直接支持
部署方案GitHub Pages免费，直接从 dist/ 目录发布
主题方案CSS 变量 + data-theme暗色默认，双主题切换，localStorage 持久化
